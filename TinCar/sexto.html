Sexto Semestre

En este semestre nos enfocamos en el desarrollo avanzado del prototipo móvil de TinCar utilizando React, implementando todas las funcionalidades principales y optimizando la experiencia de usuario para dispositivos móviles.

Implementación en React

Desarrollamos el prototipo completo de TinCar utilizando React con hooks de estado, implementando un sistema de navegación fluido entre pantallas y gestionando el estado de la aplicación de manera eficiente para una experiencia de usuario óptima.
