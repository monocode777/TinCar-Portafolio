Cuarto Semestre

En este semestre avanzamos significativamente en la experiencia de usuario con animaciones en Bootstrap, definimos la arquitectura completa del software y desarrollamos el sistema de autenticación integrado con nuestra base de datos.

Animación con Bootstrap

Implementamos animaciones y transiciones avanzadas utilizando Bootstrap 5, mejorando significativamente la experiencia de usuario. Desarrollamos interfaces dinámicas con efectos hover, transiciones suaves y componentes interactivos que hacen la plataforma más intuitiva y atractiva.

Sistema de Autenticación

Desarrollamos e implementamos el sistema completo de login y autenticación, integrado directamente con nuestra base de datos. Incluye validaciones de seguridad, manejo de sesiones, recuperación de contraseñas y diferentes niveles de acceso para usuarios y administradores.

Características del Sistema de Login

Implementamos un sistema seguro de autenticación con las siguientes características:

  • Validación de credenciales en base de datos
  • Manejo seguro de sesiones
  • Recuperación de contraseñas
  • Diferentes niveles de acceso
  • Interfaz responsive y amigable
